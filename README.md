# Eagle-Libraly-by-Morishita
個人的に作成した電子回路CADソフトEagleのライブラリ（部品データ）です。
ほとんどの部品は2013年までに作成したものですが、2017年より少しずつ追加しています。

# author
森下功啓（Katsuhiro Morishita）
熊本高等専門学校建築社会デザイン工学科助教（2018年現在）

# lisence
